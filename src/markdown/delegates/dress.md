# Dress Code

The dress code for MGSMUN 2021 is formal.

### Suggestions for Male Delegates
- Suits
- Blazers
- Smart Trousers
- Ties
- Smart Shoes

### Suggestions for Female Delegates
- Suits
- Dresses
- Skirts
- Smart Blouses or Shirts
- Smart Shoes

### Please do not wear...
- National Dress
- Jeans
- Leggings
- Trainers
- Tracksuits
