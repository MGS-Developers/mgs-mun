# Dress Code

The dress code for MGSMUN 2021 is formal.

### Suggested attire
- Suits
- Dresses
- Blazers
- Skirts
- Smart Blouses or Shirts
- Smart Trousers
- Ties
- Smart Shoes

### Please do not wear...
- National Dress
- Jeans
- Leggings
- Trainers
- Tracksuits
